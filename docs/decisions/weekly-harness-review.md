# 주간 하네스 리뷰 체크리스트

> 매주 금요일 또는 스프린트 종료 시 팀이 함께 검토한다.
> 소요 시간 목표: 15분 이내.

---

## 리뷰 진행 방법

1. 이 파일을 열고 이번 주 날짜로 새 섹션을 추가한다
2. 각 항목을 순서대로 검토하고 체크한다
3. 발견된 문제는 즉시 `docs/decisions/000-agent-mistake-log-template.md`에 기록한다
4. 하네스 업데이트가 필요하면 담당자를 지정하고 이번 주 안에 처리한다

---

## 리뷰 템플릿

```markdown

|----------|----------------|----------------------|
| controller → repository 직접 호출 | N | ✅/❌ |
| @Transactional을 controller에 붙임 | N | ✅/❌ |

### 4. 다음 주 하네스 우선순위
```
